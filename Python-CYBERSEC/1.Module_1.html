Python Fundamentals - Module 1

Fundamental Data Types

int, float, complex, str, bool, list, tuple, set, dict

1. Integer (int)

Integers are whole numbers. Basic mathematical operations can easily be performed.

print(type(1+2))  # int type class
print(1-2)
print(1*2)

2. Float

Floats represent decimal numbers. They use more memory than integers.

print(1/2)  # float data type = any kind of decimal output

3. Complex

Complex numbers are used in complex mathematical calculations (x+yi).

4. String (str)

Strings are used to store textual data.

print(type('Hello, world'))
username = "something"
password = "something"
long_string = """
This is a multi-line string.
It can span multiple lines.
This is useful for writing long texts.
"""
print(long_string)

String Concatenation

print("rexon" + " " + 'yt')

Formatted String

name = "jhonny"
age = 55
print(f'hi {name} your age is {age}')

String Indexes

selfish = '01234567'
#         01234567
# [start:stop:stepover]
print(selfish[0:8:2])
print(selfish[::-1])  # used to Reverse a String
Note: Strings are immutable and cannot be updated. Only memory override is possible.

Math Functions

print(round(2.6))  # Round off to the closest integer
print(abs(-100))   # Prints absolute values/no negative numbers

Operator Precedence

Also called operator priority, it follows the rule "PEMDAS":

  • P – Parentheses
  • E – Exponentiation
  • M – Multiplication (Multiplication and division have the same precedence)
  • D – Division
  • A – Addition (Addition and subtraction have the same precedence)
  • S – Subtraction

Miscellaneous Information

  • Variables are in snake_case (all lowercase with underscores)
  • Example variable assignments: 
    user_IQ = 29
print(user_IQ)
PI = 3.14  # Unchanging value
a, b, c = 1, 2, 3
  • Binary Conversion: 
    print(bin(5))               # prints binary output in the form of 0b____
print(int("0b101", 2))      # the binary will be stored in memory and 2 is represented as base2 which is binary
  • Statement vs Expression: 
    iq = 100      # Statement
user_Age = iq/4  # Expression
  • Augmented Assignment Operator: 
    some_value = 5
some_value += 2
print(some_value)
  • Escape Sequence: 
    weather = "It's a \"sunny\" Day Isn't it"  # whatever is in between \ is considered a string
print(weather)
